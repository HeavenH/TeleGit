####Installation:

1.Install the requirements

```
    pip install -r requirements.txt
```

2.Config Program
```
    put your token and its id chat in config.ini file
    run progam main.py
```
3.Run Progam
```
    In another instance, open the Python shell and run:
    from main import updater
    updater.start_polling()
```
